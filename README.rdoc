= Schrader 

Schrader is a tool for patrolling MediaWiki projects

== Requeriments

You need 
* ruby 1.8
* rbmediawiki gem (gem install rbmediawiki)
* sqlite3 (gem install sqlite3-ruby) Don't install sqlite3, install sqlite3-ruby instead (http://stackoverflow.com/questions/1797199/uninitialized-constant-encoding-with-sqlite3-ruby-on-windows)

== Usage

ruby schrader.rb

and point your browser to http://localhost:3000
